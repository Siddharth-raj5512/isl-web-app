# Indian Sign Language Recognition Web App

This is a deep learning project that utilizes a Convolutional Neural Network (CNN) to translate Indian Sign Language (ISL) hand gestures into text in real-time.

## Tech Stack
* **Backend:** Python, Flask, TensorFlow/Keras, OpenCV
* **Frontend:** HTML, CSS (Glassmorphism UI), JavaScript (WebRTC Video Capture)
* **Pre-processing:** Canny Edge Detection

## Deployment
This application is designed to be easily deployed on Render.com using Gunicorn.
Start command for Render: `gunicorn 4_app:app`
